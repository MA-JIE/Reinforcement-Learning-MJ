## spinning up 安装
ubuntu下: https://spinningup.openai.com/en/latest/user/installation.html <br>
其中， 
```
pip install -e .
```
命令行执行后可能会显示 runtime error <br>
解决方案: 增加国内镜像源 <br>
```
pip install -i https://pypi.douban.com/simple -e . 
```
